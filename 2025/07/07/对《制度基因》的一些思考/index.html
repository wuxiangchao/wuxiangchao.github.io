对《制度基因》的一些思考

发表于 2025-07-07 更新于 2026-01-08 阅读次数：

制度不是一天建成的，但一旦建成，它就成了所有人的宿命。

读完许成钢教授的《制度基因》，最直接的感受是：这不仅仅是一部经济学著作，更是一部关于中国历史演进逻辑的深度病理分析。

许教授在这本书中提出了一个核心概念——制度基因，通过对比东西方制度的起源与演化，解释了为什么不同文明在现代化进程中走向了截然不同的道路。

以下是对该书核心观点的整理与读书感想：

1. 核心逻辑：制度的遗传与变异

许成钢教授借鉴了生物学中的基因概念，认为制度一旦形成，就会产生强大的路径依赖。

欧洲的基因（分权）： 起源于中世纪的封建制度。君主、教会、贵族、城市之间存在权力的相互制衡，这种多中心的结构为后来的私有产权、法治和工业革命提供了土壤。

中国的基因（集权）： 起源于秦汉时期的大一统体制。核心特征是高度集权、官僚体系、行政逐级包干。这种基因在两千年的历史中不断强化，甚至在现代转型中依然发挥着底层代码的作用。

2. 深度剖析：为什么是向性而非偶然？

书中一个非常震撼的观点是：制度的演化具有方向性。

行政包干制与准司法独立
许教授详细分析了中国历史上的行政包干制。中央政府并没有能力直接管理每一个县，因此采取了将任务逐级下放的方式。这种体制在承平时期效率极高，但在面对外部冲击或需要制度创新时，往往因为缺乏独立的司法体系和激励机制，陷入一收就死，一放就乱的怪圈。

工业革命的缺位
通过对比，书里解答了著名的李约瑟难题。西方之所以爆发工业革命，是因为其制度基因允许自下而上的创新和对私有财产的绝对保护；而中国的制度基因倾向于将所有资源纳入行政管控，创新往往被视为对统治秩序的潜在威胁。

3. 感想与启示：改革的深水区到底是什么？

3.1 制度决定了技术的天花板

很多人认为技术可以解决一切问题，但《制度基因》警示我们：如果没有制度的配套，单纯的技术引进只能实现短期的追赶，无法实现长期的领先。 制度基因决定了创新的激励机制，也决定了社会资源的分配效率。

3.2 路径依赖的沉重感

读完此书，会感到一种深深的紧迫感。制度基因的遗传是非常顽固的。即使表面上的经济数据在增长，如果底层的权力运作模式和产权保护逻辑没有发生根本性改变，这种增长往往是脆弱的。

3.3 全球视野下的中国位置

许教授通过全球比较，提醒我们不要陷入特殊论的误区。尽管每个国家都有自己的国情，但关于分权、法治、竞争的普世性规律，在任何制度基因面前都是有效的。

4. 总结

《制度基因》是一本让人清醒的书。它打破了对短期经济繁荣的幻觉，迫使读者去思考那些隐藏在水面之下的、跨越千年的制度惯性。
